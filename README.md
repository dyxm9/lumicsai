# lumicsai
LUMICS é uma plataforma educacional e tecnológica que analisa o impacto ambiental da Inteligência Artificial. O projeto busca conscientizar usuários sobre o consumo energético da IA, promover práticas mais sustentáveis e oferecer ferramentas educativas e visuais que ajudam a entender como a tecnologia pode evoluir de forma responsável.
